Hey, I'm Clara Alder and I do web stuff.

I am currently available for freelance or full-time work, remotely or in the Brighton area. Please get in touch!

I specialise in HTML, CSS/SASS and JS. I also have a background in game development and graphic design.

I've enjoyed attending Codebar—a non-profit initiative which runs programming workshops—for many years, both a student and as a tutor.

Websites

  • Bookkeeping website.

    This Bookkeeping website was built without a CMS but after it was finished I added in some extra CSS selectors to make it editable in PageLime (now discontinued) after it was finished.

  • Health Promotion Website.

    I used Craft CMS + SCSS + jQuery to create this website for a Health Promotion Practitioner. Building it using Craft CMS meant it she could edit the text and images without having to edit any code herself. Building the site using Craft CMS wass great as it uses the Twig templating system, allowing for a lot of flexibility.

    The website is no longer active but the above image links to a web archive of the site. The archived version of the site is in a finished state other than some duplicate paragraphs and multicolour text from when we were playing around with layouts.

Game Jams

I've taken part in six game jams. The following are some of the games that I was involved in making:

  • The Carrier Dodo Company

    Congratulations!

    It is your first day as a trainee Post-Dodo, working at The Carrier Dodo Company. It is, for some reason, less popular than Hiddletims Elite Carrier Pigeon Service; but I am sure you are excited to be part of the team, and you look very smart.

    Your first (and only) assignment is to collect mail from the postboxes, and deliver them to the houses indicated by the arrow.

    Click above to download and play The Carrier Dodo Company!

    Update! Thanks Jupiter for writing this nice piece about the game on Indie Games +

  • Duck's Cosy Home

    Duck's grandpa has passed away and duck is moving back to where he grew up to sort through belongings and memories.

    Update! Thanks Jupiter for writing this nice piece about the game on Indie Games +

  • Mr Boots Finds His Way

    Mr Boots is walking to his grandpa's house on his own for the first time. He has written down the instructions that his grandpa told him.

  • Brituals

    ‘Brituals’ is a ‘British-themed cross-dimensional social awkwardness simulacrum’ created at the 2016 Global Game Jam. The jam's theme was 'ritual'.

  • Wave or Waive

    At another game jam the theme was 'waves'. I co-created 'Wave or Waive', a game about this common dilemma:

    Someone is waving at you. Or are they waving at someone behind you? It would be embarrassing if you waved back like an idiot. This is a difficult choice and you only have a split second to make it.

    Click above to download and play 'Wave or Waive'!

  • Mallardies

    A three-player duck co-opera (co-op opera) game. Every player uses just one button each, as with our previous game Wave or Waive. Quack along to Nessun Dorma. Be the new Drake. Be the best duck operatist that you can be.

    Click above to download and play 'Mallardies'!

Arcade Box

An arcade box built to play 'Wave or Waive' and 'Mallardies'. Both games have the same unusual control scheme of one button per-person. This was built using an Arduino Leonardo clone and allows for three people to stand beside one another, each with their one button.

  • The arcade box was part of the Pop Up Arcade 2019!
